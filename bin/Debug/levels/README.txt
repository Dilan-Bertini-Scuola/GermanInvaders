Per creare un livello vanno mantenute le seguenti regole:

1) Deve avere un confine 20x20
2) Deve contenere almeno un H
3) Il punto di spawn deve essere indicato con X

# = Ostacoli anti-carro/confine ( Non Distruggibili )
H = Carri nemici
X = Punto di spawn
/ = Ostacoli ( Distruggibili )

Tutti gli altri caratteri verranno considerati come nulli
